# 算法导论

## 先行条件

- 计算机数学（离散数学）
  - 假设归纳

## 课程描述

1. 介绍计算问题的数学模型，常见算法、算法范式，和用来解决这些问题的数据结构
2. 强调算法与编程之间的关系
3. 介绍这些算法问题的基本性能度量和分析技术

## 参考书

**CLRS**——《算法导论》（可选）

## 适合人群

- 初学者（非快餐式进食目的， 事实性的知识刻板接受，不在乎底层的设计思想）
- 想学《算法导论》遇到瓶颈的学习者
