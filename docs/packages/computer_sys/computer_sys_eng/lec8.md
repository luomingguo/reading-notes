# Lec 8 计算机网络介绍

![截屏2024-07-04 16.17.54](http://14.103.135.111:49153/i/66865ab93619e.png)

随着系统的发展，我们需要考虑，如何将一组链路加入到网络中。

| 分层   | 作用                                         |
| ------ | -------------------------------------------- |
| 应用层 | 实际产生流量的东西                           |
| 传输层 | 共享网络，可靠传输(或者不可靠)               |
| 网络层 | 命名、定位、和路由                           |
| 链路层 | 两个相连接点的通信。比如BLT,ethernet, 802.11 |
